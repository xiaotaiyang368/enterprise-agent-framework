# L0 决策智能体规范

## 概述

决策智能体（Decision Agent）是企业级元智能体，位于三层架构的最顶层。它拥有全量企业业务模型视图，感知一切内外部信号，通过顶级大模型引擎进行推理、规划、决策和工具调用，编排整个组织的运转。

## 认知范围

决策智能体维护企业业务的全量五要素视图：

- **事件**: 全市场信号（政策变化/竞品动态/行业趋势）+ 全内部事件流（所有 L1/L2 的汇报和告警）
- **流程**: 企业级 SOP 之间的编排关系、依赖图、冲突仲裁规则
- **对象**: 企业全量实体视图（组织/人员/资产/资金/社会关系）
- **规则**: 战略约束 + 合规红线 + 资源分配策略
- **行为**: 调度/编排/决策类高层操作

## 关键能力

### 1. 战略认知

从外部市场信号和内部运营数据中提炼战略洞察，将模糊的目标转化为可执行的计划。

```
输入: "本季度我们要降低贸易风险敞口"
输出: [
  { action: "强化L1采购运营智能体的三流比对规则", priority: "high" },
  { action: "调度L2秘书助手安排风控培训", priority: "mid" },
  { action: "通知L1财务运营智能体冻结高风险供应商付款", priority: "high" }
]
```

### 2. L1 编排

根据任务类型和目标，选择最合适的 L1 运营智能体，或将跨域任务拆解后编排多个 L1 协作。

### 3. 资源调度

在企业范围内动态调度资源（人力/资金/计算资源），仲裁跨域冲突。

### 4. 反馈闭环

收集 L1/L2 的执行反馈，评估实际效果，不断优化战略决策质量。

## 与 E5M Engine 的关系

决策智能体不直接执行业务逻辑。它通过调用 E5M Engine 的 API 来：

1. 读取五要素模型（认知世界）
2. 下发执行指令（触发事件/启动流程）
3. 获取执行反馈（查询状态/阅读审计日志）
